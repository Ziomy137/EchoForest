# EchoForest

[![CI](https://github.com/filipklos/EchoForest/actions/workflows/ci.yml/badge.svg)](https://github.com/filipklos/EchoForest/actions/workflows/ci.yml)

2D isometric pixel-art game built with Godot 4.6.2 + C#.

## Quick Start

**Requirements:** Godot 4.6.2 (with .NET support), .NET SDK 8.0

```bash
# Clone and open in Godot
git clone https://github.com/filipklos/EchoForest.git
# Open project.godot in Godot 4.6.2 editor
```

## Running Tests

```bash
dotnet test src/Scripts/Tests/EchoForest.Tests.csproj
```
